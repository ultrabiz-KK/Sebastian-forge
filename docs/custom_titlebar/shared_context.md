# 共有コンテキスト: カスタムタイトルバー

## 概要
- タイトルバーをネイティブからカスタムUIに変更しました。
- 今後ウィンドウ操作に関する機能を拡張する場合は、`src/components/layout/TitleBar.tsx` を編集してください。

## 留意点
- **パーミッション**: Tauri v2では、新しいウィンドウ操作（例：最前面表示の切替など）を追加する場合、`src-tauri/capabilities/default.json` に権限を追記する必要があります。
- **ドラッグ領域**: ボタン以外の領域には `data-tauri-drag-region` を付与し、操作性を維持してください。
- **テーマ変数**: タイトルバーの色は `--titlebar-*` 変数で管理されています。
